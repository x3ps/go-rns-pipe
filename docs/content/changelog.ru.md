## Не выпущено

### Обновление до Go 1.26
- Модуль обновлён до Go 1.26

### Расширение E2E-тестового набора
- Расширены тесты передачи ресурсов с метаданными и различными размерами
- Комплексные тесты паритета и точности для побайтовой совместимости на уровне протокола
- Тесты жизненного цикла соединений и нагрузочные тесты (много пакетов, много параллельных соединений)
- Тесты упорядоченности сообщений в каналах и тесты с большими данными
- Общие вспомогательные функции conftest для Python E2E-тестов (`establish_link`, `resolve_packet_dest`)

### Улучшения тестовой инфраструктуры
- Тесты переподключения и параллельного `Receive` (`pipe_test.go`)
- Тесты фаззинга и граничных условий для HDLC (`hdlc_test.go`)
- Тесты потокобезопасности для `Decoder`
- Защита `ErrNoHandler`: `Start` возвращает ошибку, если `OnSend` не зарегистрирован до запуска
- Интеграционные тесты обновлены с учётом требования `ErrNoHandler`

### Исправления ошибок
- **утечка goroutine:** заменён `time.After` на `time.NewTimer` для предотвращения утечек goroutine таймера
- **потокобезопасность HDLC:** добавлена mutex-защита для `Decoder`

### UDP-транспорт
- Исправлен порядок шагов при запуске транспорта
- Добавлено правильное управление жизненным циклом context
- Инъекционный шов `pipeConfig` для подстановки пар `io.Pipe` в тестах
- Тест закрытия пайпа

### TCP-транспорт
- Регистрация обработчика `OnSend` перед `Start` для соответствия требованию `ErrNoHandler`

### Документация
- Добавлен раздел сторонних компонентов с лицензиями (`NOTICE`)
